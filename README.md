# 👗 Luna Vogue Boutique Website

## 📌 Project Overview

**Luna Vogue Boutique** is a modern fashion boutique website designed to showcase stylish clothing collections and provide information about the brand.
The website presents products, brand details, and contact information through a clean and visually appealing interface.

This project demonstrates **front-end web development skills using HTML and CSS** and is suitable for showcasing a fashion brand online.

---

# 🎯 Project Objectives

* Create a stylish and user-friendly fashion boutique website
* Display clothing collections attractively
* Provide information about the boutique brand
* Allow customers to easily contact the boutique

---

# 🖥️ Website Pages

| Page                             | Description                                                 |
| -------------------------------- | ----------------------------------------------------------- |
| **Home (index.html)**            | Landing page introducing the boutique and featured products |
| **About (about.html)**           | Information about the brand and its story                   |
| **Collection (collection.html)** | Displays fashion items available in the boutique            |
| **Contact (contact.html)**       | Contact details and ways to reach the boutique              |
